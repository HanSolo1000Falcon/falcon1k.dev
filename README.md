# falcon1k.dev

> Personal portfolio and developer website for **HanSolo1000Falcon** — built with React, TypeScript, and Vite, deployed on Cloudflare Pages.

**Live site:** [falcon1k.dev](https://www.falcon1k.dev)

---

## Tech Stack

| Layer | Technology                                                       |
|-------|------------------------------------------------------------------|
| Framework | [React](https://react.dev/)                                      |
| Language | [TypeScript](https://www.typescriptlang.org/)                    |
| Bundler | [Vite](https://vitejs.dev/)                                      |
| Deployment | [Cloudflare Pages](https://pages.cloudflare.com/) (via Wrangler) |
| Linting | [ESLint](https://eslint.org/)                                    |
